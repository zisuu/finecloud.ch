ShellCheck

Beschreibung

ShellCheck ist ein Tool, das Warnungen und Vorschläge für Bash/Sh-Shell-Skripte liefert. Das Tool kann verwendet werden für statische Code Analyse.

CI Pipeline

Idealerweise wird das Tool in einer CI Pipeline integriert. Hier ein Beispiel:

name: Shellcheck

on:
  pull_request:
  push:
    branches:
      - main

jobs:
  shellcheck:
    name: Github Actions
    runs-on: ubuntu-latest
    steps:
    - uses: actions/checkout@v3
    - name: Run Shellcheck
      uses: ludeeus/action-shellcheck@master
      with:
        check_together: 'yes'
      env:
        SHELLCHECK_OPTS: --shell bash
